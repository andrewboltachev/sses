Начиная с версии 11.04 «Natty Narwhal», которая вышла 28 апреля 2011 года[25], стандартная для Ubuntu Desktop Edition среда рабочего стола GNOME заменена на Unity[26][27][28], разрабатываемую компанией Canonical, а в следующих версиях X.Org Server будет заменён на Wayland[29]. Также изменена система наименований дистрибутивов Ubuntu: Ubuntu Netbook Edition и Ubuntu Desktop Edition слиты в одну версию, называемую просто «Ubuntu» (без слова «Edition»), в свою же очередь Ubuntu Server Edition тоже лишился этого слова и теперь это просто «Ubuntu Server»[30]. Предсказуемые изменения, так как оболочка Unity, которая использовалась в Ubuntu Netbook Edition 10.10, перекочевала на Desktop версию Ubuntu начиная с версии 11.04 и стала в ней графической оболочкой по умолчанию. Теперь стандартный образ Ubuntu стал мультиплатформенным и может устанавливаться как на нетбуках, так и на ноутбуках и десктопных компьютерах.
